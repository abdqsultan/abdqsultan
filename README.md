## About Me

I build web applications and websites from scratch — no templates, no page builders, no shortcuts.

Every project I deliver is purpose-built: handcrafted PHP, CSS, and JavaScript tailored precisely to the client's requirements. I specialize in complex, enterprise-grade solutions that demand clean architecture, multilingual support, and long-term maintainability.

Under the **LanguMax** brand, I work with NGOs, businesses, and organizations that need more than what off-the-shelf tools can offer.
